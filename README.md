# OrdersManageSystem
Этот проект реализует фронтенд информационной системы для управления процессами мебельной компании. Диаграмма Ганта отображает задачи по сборке и доставке мебели, позволяя наглядно планировать и контролировать этапы обслуживания клиентов.

## О проекте

Этот проект представляет **фронтенд информационной системы для мебельной компании**, которая занимается **сборкой и доставкой мебели**.  
Система позволяет:

- Визуально формировать очередь оказания услуг с помощью **диаграммы Ганта**.  
- Отслеживать задачи по сборке и доставке мебели.  
- Назначать исполнителей (экипажи) на конкретные задачи.  
- Управлять временными интервалами (дни/недели) и рабочими сменами.  
- Интерфейс различается в зависимости от роли пользователя (администратор, логист, продавец, грузчик-сборщик).  

> ⚠️ Backend реализован минимально и не включён в этот репозиторий. Основная цель — демонстрация фронтенда.

---

## Цель проекта

Автоматизировать процесс расчета длительности выполнения сборки и доставки мебели и предоставить **инструмент визуальной оценки загрузки сборщиков-грузчиков**, чтобы:

- Сократить перепланирования.  
- Улучшить организацию работы сотрудников.  
- Повысить качество обслуживания клиентов.

---

## Функционал фронтенда

### Администратор
- Назначение ролей пользователям.  
- Просмотр общего состояния системы.  

### Логист
- Список договоров с мебелью на складе (готовы к выдаче) 
- Диаграмма Ганта для планирования доставки и сборки.  
- Возможность добавлять нестандартные задачи (выезды на замену, особые заказы).  
- Управление экипажами и днями (выходные, рабочие смены).  

### Продавец
- Список договоров, автором которых он является.  
- Возможность размещать свои блоки на диаграмме и добавлять комментарии к ним.  

### Грузчик-сборщик
- Просмотр только своих задач на диаграмме Ганта.  
- Мобильная версия сайта.  

⚠️ В этом репозитории демонстрируется только интерфейс логиста. Другие роли (администратор, продавец, грузчик-сборщик) не включены.

## Технологии

- **HTML5, CSS3, JavaScript (ES6)** — фронтенд.  
- Использован **гeометрический гротеск Mont Blanc** для интерфейса.  
- **Диаграмма Ганта** реализована через кастомные JS-модули и CSS-блоки.  

> Backend (PHP/MySQL) не включён.
